Before running the plot_bic.m:
	load the bicoherence information in bic_info.mat for each patient in the corresponding folder
	bic_info.mat includes info_EZ for dH pattern and info_PZ for non-dH pattern

Before running the plot_tf_prop.m:
	load the propagation model in prop.mat for each patient in the corresponding folder
	prop.mat includes EZtoPZ (fitting PZ by filtering EZ) and PZtoEZ (fitting EZ by filtering PZ)
